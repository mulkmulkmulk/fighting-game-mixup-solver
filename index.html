How to use

1. Pick the matrix size, up to 10×10.
2. Edit row and column move names directly in the matrix headers or in the text boxes above.
3. Enter payoffs in each cell. Red values are your payoffs, blue values are the opponent's payoffs.
4. Press Solve to compute the optimal mixed strategy frequencies and expected values.
